# Expense Tracker with AI Suggestions

This project is a full-stack web application that allows users to track their daily expenses and get personalized financial suggestions powered by AI.

## Features

- Add, edit, and delete expense entries
- Visual representation of spending patterns
- AI-generated suggestions to improve financial planning
- Frontend built with React, Backend built with Flask

## Tech Stack

- **Frontend**: React.js, HTML, CSS
- **Backend**: Python, Flask
- **Database**: SQL (SQLite or MySQL)
- **Communication**: Axios (for frontend-backend integration)

## How to Run

### Frontend

```bash
cd frontend
